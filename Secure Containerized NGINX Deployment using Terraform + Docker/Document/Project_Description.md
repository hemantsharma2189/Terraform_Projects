# Project Description

This project demonstrates how Terraform can automate local container deployment using Docker.

Instead of manually running Docker commands, Terraform provisions an NGINX container, creates a custom Docker network, maps the container port to localhost, applies restart behavior, limits container resources, and serves a custom HTML web page.

## Key Features

- Terraform-based Docker automation
- NGINX container deployment
- Custom Docker network
- Local port mapping
- Read-only mounted HTML file
- Restart policy
- CPU and memory limits
- Infrastructure as Code workflow
